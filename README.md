# The Bot API

A simple sample [Spring Boot](http://projects.spring.io/spring-boot/) application, using [Liquibase](http://www.liquibase.org/) to control the database.

## Requires

- Java (>= 1.8)
- Apache Maven (>= 3.3.9)
- Docker (>= 1.11.2)
- Docker Machine (>= 0.7.0)
- Docker Compose (>= 1.8.0)
